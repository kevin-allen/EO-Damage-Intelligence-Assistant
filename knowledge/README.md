# Knowledge Corpus

Hand-curated markdown documents indexed by ChromaDB and retrieved at report-
generation time. See D3 in `software_requirements.md`.

**Target:** ~12 documents, 300–800 words each, covering:

- Building damage taxonomies and assessment conventions
- EO imagery limitations and uncertainty (resolution, registration, occlusion, lighting)
- Disaster-response prioritization frameworks
- Urban infrastructure risk
- Per-disaster-type guidance:
  - `hurricane.md`
  - `wildfire.md`
  - `flood.md`
  - `earthquake_tsunami.md`

Adapted from public FEMA, USGS, ESA, and xView2-paper sources. Documents
are not yet written — to be added during the implementation phase.
